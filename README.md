## 介绍

vueblog-app，基于 Springboot + Vue 技术构建。

- 一个简单的博客系统。
- 基于spring boot和vue的前后端分离的项目

## 技术选型

- 核心框架：Spring Boot2.7.5
- 数据库层：Mybatis/Mybatis-Plus
- 缓存：Caffeine/Redis
- 前端：Vue2.x.js/Element Ui
- 数据库：Mysql5.7.33
